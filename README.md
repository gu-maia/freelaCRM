# freelaCRM

 * Client Management.
 * Client Portal.
 * Quotation Request.
 * Expenses.
 * Estimates.
 * Project Management.
 * Project Tasks.
 * Lead Management.

 * Project Milestones.

The goal is to create a Management platform for freelancers, completely free and open source.
